01

Sentence Structure

S   Subject
+
V   Verb
+
O   Object (optional)
+
Time word
Key rule: Indonesian follows Subject-Verb-Object order — the same as English. This makes building sentences immediately intuitive.
Time words like besok (tomorrow) and kemarin (yesterday) go at the beginning or end of the sentence. Putting them at the front adds emphasis; at the end is more neutral.
No verb conjugation for tense. The verb never changes form. Tense is expressed through time words and markers: sudah (already/past), sedang (currently), akan (will). Add kemarin and it becomes past. Add besok and it becomes future.
02

Pronouns

Key Note
Indonesian has formal and informal pronouns. Saya (formal I) and aku (informal I) are both correct — context determines which to use. With strangers, elders, or in professional settings, use saya and Anda. With friends and peers, aku and kamu are natural. Dia covers he, she, and it — Indonesian does not distinguish gender in the third person.

Formal vs Casual Verbs
In casual spoken Indonesian you will hear bare base verbs: dia makan (she eat). In formal Indonesian or writing, the me- prefix is added: dia memakan. Both are understood and correct in context. As a beginner, use base verbs in conversation — native speakers do too.

Negation — Tidak vs Bukan
Indonesian has two negation words. Tidak negates verbs and adjectives — insert it directly before the verb: Saya → Saya tidak → Saya tidak makan. Bukan negates nouns and identity: Ini bukan nasi (this is not rice). In casual speech, nggak or gak replaces both. Getting tidak/bukan right marks you as a careful speaker.

Di mana vs Ke mana
This distinction trips up many learners. Di mana = where is something located (static). Ke mana = where is something going (direction). Di mana rumahmu? (Where is your house?) vs Ke mana kamu pergi? (Where are you going?). The preposition (di = at, ke = to) carries the meaning.

07

Practice Drills

Drill these until automatic
  • Say the core pattern out loud: Saya makan kemarin. Saya tidak makan kemarin. Saya makan besok. The verb never changes — only the time word and negation shift.
  • Swap the subject through all pronouns: Saya makan. Kamu makan. Dia makan. Kita makan. Mereka makan. Notice the verb stays identical every time.
  • Build the ordering sentence from memory: Saya mau pesan nasi campur satu. Then change the number: dua, tiga, empat.
  • Practice the where-question with both forms: Di mana kamu makan kemarin? Then answer: Saya makan di rumah kemarin. (I ate at home yesterday.)
  • Negate three sentences using tidak and three using bukan. Check: is it negating a verb, or a noun?
  • Swap verbs through the same sentence frame: Saya minum kemarin. Saya masak kemarin. Saya pergi kemarin.
  • Practice both registers back to back: Saya tidak mau (formal) → Aku nggak mau (casual). Same meaning, completely different feel.

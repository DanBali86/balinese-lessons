How to Add a New Lesson

Step 1. Upload the new lesson HTML file to your GitHub repository (e.g. tutor-lesson-2.html

Step 2. Open index.html in GitHub and click the pencil ✏️ edit icon. Find the "Coming soon" card for Lesson 02 and replace it with this pattern:

<a class="lesson-card" href="tutor-lesson-2.html">

Step 3. Update the title, description tags, and the stats numbers at the top of this page to reflect the new lesson count.

Step 4. Add a new "Coming soon" placeholder card at the bottom of the list so there's always a slot ready for the next session.

Step 5. Commit changes — the hub updates live within 60 seconds.
